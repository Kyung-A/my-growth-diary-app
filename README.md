# 나의 성장일기📖

✍목표를 설정하고 일기를 쓸 수 있는 앱

### 개요

React Native와 expo를 이용해 디자인, 퍼블리싱한 나의 성장일기 App입니다.

### 구현내용

1. 퍼블리싱 작업
2. Expo 사용
3. React Navigation의 Authentication flows 를 참고하여 인증 상태에 따라 화면 분기
4. React Native Navigation 사용
5. 캘린더, 체크박스, date picker 사용
---

### 📸 화면
이미지 순서대로입니다.
1. 로딩화면
2. 회원정보 입력
3. 목표 설정
4. 메인화면
5. 일기보기
6. 일기작성
7. 목표 모아보기
